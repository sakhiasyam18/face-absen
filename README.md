1. Set up virtual environment python di terminal command = python -m venv venv
2. aktifkan virtual environment di terminal command = ./venv/Scripts/activate
3. install requirements di terminal command = pip install -r requirements.txt
4. running di terminal = python run_server.py pilih  http://127.0.0.1:5000
5. untuk hapus data jalankan file delete_user.py dengan command = python delete_user.py
